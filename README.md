# StackBoxAI 📦 (Employee App)

> **Note:** The source code for this project is hosted in a private organization repository. This repository serves as a portfolio showcase of the architecture, features, and my contributions to the project.

## Overview
StackBoxAI is a comprehensive warehouse management solution designed to streamline logistics tracking. 

## My Contributions
* Developed and integrated automated QR code generation and scanning features for rapid inventory logging.
* Implemented real-time object detection models directly into the mobile workflow.
* Authored comprehensive project documentation covering system architecture, API integration, and setup instructions.

## Tech Stack
* **Frontend:** Kotlin, Jetpack Compose
* **Backend & Database:** Supabase
* **AI & Computer Vision:** YOLOv8, Google ML Kit
* **Version Control & Tools:** Git
